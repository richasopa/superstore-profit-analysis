#  Superstore Profit Analysis

##  Overview
This project analyzes retail data to understand factors affecting profit margin and identify business risks.

##  Objective
- Analyze profit distribution
- Identify high-risk categories
- Predict profit margin using machine learning

##  Tools Used
- Python (Pandas, Seaborn, Scikit-learn)
- Jupyter Notebook

##  Key Insights
- Profit margins are mostly positive with a long tail of losses
- Technology is the most stable and profitable category
- Furniture shows consistent underperformance
- Office Supplies has extreme loss outliers

##  Model
- Random Forest Regressor
- R² Score: 0.9118
- MAE: 0.0970

##  Outcome
This project demonstrates how data analysis and machine learning can support business decisions.
